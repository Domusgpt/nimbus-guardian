How It Works

1

Install

One command to install Nimbus globally

npm install -g nimbus-guardian
2

Setup

Quick 2-minute interactive setup

nimbus setup
3

Scan

Scan your project for issues

nimbus scan --ai
4

Deploy

Deploy with confidence!

nimbus pre-deploy

What Nimbus Catches

Oh Shit Level

  • Exposed API keys & secrets
  • .env files in git
  • Hardcoded passwords
  • AWS/Stripe live keys

Bad Idea Tier

  • Missing security headers
  • Vulnerable dependencies
  • Docker root user
  • Missing .gitignore patterns

You Should Probably...

  • No .env.example
  • Missing compression
  • Outdated dependencies
  • No caching strategy
